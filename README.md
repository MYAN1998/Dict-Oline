# Dict-Online

#### 介绍
使用python实现在线词典功能;拥有以下功能:登录,注册,退出,查单词,历史记录,注销;数据存储:MySQL数据库;并发模型:多进程,TCP并发;

#### 软件架构
软件架构说明
~~~
注册
    客户端 :  发送请求,等待响应,根据响应具体处理;
    服务端 :  接收请求,判断是否可以注册,根据结果发送响应;
登录:与注册类似
查单词
历史记录
~~~
#### 使用教程

1.  首先运行server.py
2.  再运行client.py
